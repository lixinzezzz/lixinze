1. 攻击技术

    a. 拒绝服务攻击（denial-of-service attack，DoS）
    拒绝服务攻击，亦称洪水攻击，其目的在于使目标电脑的网络或系统资源耗尽，使服务暂时中断或停止，导致其正常用户无法访问。
    分布式拒绝服务攻击，指攻击者使用两个或以上被攻陷的电脑作为“僵尸”向特定的目标发动“拒绝服务”式攻击。

    b. SQL注入攻击
    服务器上的数据库运行非法的 SQL 语句，主要通过拼接来完成。
    使用参数化查询解决。例如Java中的 PreparedStatement 是预先编译的 SQL 语句，可以传入适当参数并且多次执行。由于没有拼接的过程，因此可以防止 SQL 注入的发生。

    c. 跨站请求伪造（Cross-site request forgery，CSRF）
    跨站请求伪造（Cross-site request forgery，CSRF），是攻击者通过一些技术手段欺骗用户的浏览器去访问一个自己曾经认证过的网站并执行一些操作（如发邮件，发消息，甚至财产操作如转账和购买商品）。由于浏览器曾经认证过，所以被访问的网站会认为是真正的用户操作而去执行。CSRF利用的是网站对用户浏览器的信任。
    可以通过检查Referer首部的字段，要求请求来源的地址在同一个域名下，和输入验证码，让用户知道自己正在做的操作来解决。

    d. 跨站脚本攻击（Cross-Site Scripting, XSS）
    将代码注入到用户浏览的网页上，这种代码包括HTML和JavaScript。
    可以通过过滤特殊字符的方式来解决，比如富文本编辑器通常采用 XSS filter 来防范 XSS 攻击，通过定义一些标签白名单或者黑名单，从而不允许有攻击性的HTML代码的输入。form和script等标签都被转义，而h和p等标签将会保留。

2. RSA

    a. 找到大素数p, q
    b. n = pq
    c. 找到e, s.t. e = 1 mod(phi(n)) 
    d. e^d = 1 mod (phi(n))
    e. (e, n) <- 公钥
    f. (d, n) <- 私钥
    加密: m^e mod(n)
    解密: c^d mod(n)
